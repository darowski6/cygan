Team Planner

This app stores your data in Google Drive as a JSON file. You need a free Google Cloud OAuth Client ID. Follow the steps below — it takes ~3 minutes.

Step 1 — Create a Google OAuth client

  1. Go to console.cloud.google.com
  2. Create or select a project
  3. Enable Google Drive API under APIs & Services
  4. Go to Credentials → Create Credentials → OAuth client ID
  5. Choose Web application
  6. Under Authorised JavaScript origins add https://darowski6.github.io
  7. Copy the Client ID

Also go to OAuth consent screen, add your Google account as a test user, set app to External.

Step 2 — Enter your Client ID

Your Client ID is saved in your browser. Nothing is sent anywhere except Google's OAuth servers.
