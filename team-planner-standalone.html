This app stores your data in Google Drive as a JSON file. To connect it, you need a free Google Cloud OAuth client ID. Follow the steps below — it takes about 3 minutes.

Step 1 — Create a Google Cloud OAuth client

  1. Go to console.cloud.google.com
  2. Create a new project (or select an existing one)
  3. Go to APIs & Services → Enabled APIs and enable Google Drive API
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID
  5. Choose Web application
  6. Under Authorised JavaScript origins add the URL where you'll host this file (e.g. https://yourusername.github.io — or http://localhost for local use)
  7. Click Create and copy the Client ID

Also go to OAuth consent screen, add your own Google account as a test user, and set the app to External.

Step 2 — Enter your Client ID

Your Client ID is saved in your browser's local storage. Nothing is sent anywhere except Google's own OAuth servers.
